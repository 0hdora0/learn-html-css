CSS 기초

CSS를 HTML에 적용하는 세 가지 방법

External Style Sheet

CSS 파일을 별도로 생성하여 HTML 문서에 연결하는 방식으로 <link> 요소를 사용하는 방법과 @import 명령을 사용하는 두 가지 방식이 있습니다.

Embedded Style Sheet

별도로 CSS 파일을 생성하지 않고 HTML 파일 내에 CSS 코드를 직접 포함하여 스타일을 적용하는 방식으로 CSS 코드는 <style> 요소 내에 선언합니다.

Inline Style Sheet

특정 HTML 요소에 style 속성을 사용하여 CSS 코드를 선언하는 방법입니다. 그러나 이 방법은 구조와 표현의 분리라는 관점에서 바람직하지 않은 면을 가지고 있습니다. 특히 Inline 방식의 스타일은 선택자의 우선순위가 가장 높기 때문에 스타일의 재정의가 어렵거나 불가능한 경우가 발생할 수 있으므로 사용에 주의가 필요합니다.
